## 第7页：技术实现

```
标题：自托管 LLM 引擎，全栈自主可控

┌──────────┬────────────────────────┬──────────────────────┐
│  层次    │     技术选型           │      用途            │
├──────────┼────────────────────────┼──────────────────────┤
│ 代理层   │ 清小搭平台             │ 用户入口 + 标准协议   │
│ LLM 引擎 │ DeepSeek (deepseek-chat)│ 对话理解 + 工具调度   │
│ 后端     │ Python Flask           │ API 服务（OpenAI 兼容）│
│ 视觉     │ DeepSeek 多模态        │ 图片识别分析          │
│ 语义     │ SiliconFlow BGE        │ 模糊搜索（768维向量）  │
│ 搜索     │ DeepSeek web_search    │ 联网搜索辅助生成介绍   │
│ 存储     │ JSON 文件              │ 商品数据持久化        │
│ 部署     │ 阿里云 ECS + Nginx     │ 公网服务              │
└──────────┴────────────────────────┴──────────────────────┘

为什么选这些？
    · OpenAI 兼容协议，清小搭零适配接入
    · Function Calling 让 LLM 自主调度 8 个工具
    · API Key 已配置，全部可用（DeepSeek + SiliconFlow）
    · 阿里云 ECS 7×24 运行
```
